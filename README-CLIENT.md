# Guide d'utilisation — Site Casa Pronto

Bienvenue ! Ce guide vous explique comment **modifier votre site vous-même**, sans aucune connaissance technique. Vous pouvez changer votre menu, vos prix, vos horaires et vos photos quand vous voulez.

---

## 🔑 Se connecter à l'administration

1. Allez sur **votre-site.fr/admin**
2. Cliquez sur **« Se connecter avec GitHub »**
3. Entrez vos identifiants (créés lors de la mise en place)

Vous arrivez sur une interface simple avec deux sections :
- **Infos du restaurant** (coordonnées, horaires, photos)
- **Menu** (vos catégories et plats)

---

## 🍽️ Modifier le menu

1. Cliquez sur **Menu** dans le menu de gauche
2. Choisissez une catégorie (Burgers, Boissons…) ou cliquez sur **« New Catégorie »** pour en ajouter une
3. Pour chaque plat, vous pouvez modifier :
   - le **nom**
   - la **description**
   - le **prix**
   - une **photo** (optionnel)
   - la case **« populaire ★ »** pour mettre un plat en avant
4. Pour **ajouter un plat** : cliquez sur **« Add Plats »** en bas de la liste
5. Pour **supprimer un plat** : cliquez sur la corbeille à côté du plat

➡️ **N'oubliez pas de cliquer sur « Publish » (en haut à droite) pour enregistrer.**

---

## 🕒 Modifier horaires, téléphone, adresse

1. Cliquez sur **Infos du restaurant** → **Coordonnées & horaires**
2. Modifiez ce que vous voulez
3. Cliquez sur **Publish**

---

## 📸 Ajouter des photos à la galerie

1. **Infos du restaurant** → **Galerie photos**
2. Cliquez sur **« Add Photos »**
3. Uploadez votre image et ajoutez une légende
4. Cliquez sur **Publish**

---

## ⏱️ Combien de temps avant que ça s'affiche ?

Après chaque **Publish**, votre site se met à jour automatiquement en **30 secondes à 2 minutes**. Rechargez la page pour voir le résultat.

---

## ❓ Conseils

- **Les photos** : utilisez des images au format paysage, idéalement moins de 1 Mo (vous pouvez les compresser sur un site comme squoosh.app).
- **Vous avez fait une erreur ?** Aucun risque de « casser » le site : tout est sauvegardé et réversible.
- **Une question ?** Contactez votre prestataire.

Bon service ! 🍕
